# Install Unity in different ways

For more detail, please read [Unity Manual](https://docs.unity3d.com/Manual/GettingStartedInstallingUnity.html)

## Install Unity using Unity Hub

1. Download and install Unity Hub (lastest version)
2. Open Unity Hub
3. Install Unity Editor (recomemend lastest LTS version)

## Installing Unity from the command line

Follow this [guide](https://docs.unity3d.com/Manual/InstallingUnity.html)

## Installing Unity offline without the Hub

Follow this [guide](https://docs.unity3d.com/Manual/DeployingUnityOffline.html)
