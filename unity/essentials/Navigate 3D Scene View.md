# Navigate 3D Scene View

- <kbd>Alt</kbd> + hold and move <kbd>Left Mouse</kbd>: Rotate around the focal point (Current Selected Game Object Position).

- Hold and move <kbd>Right Mouse</kbd>: Roate(Look) around current point (Current your view point in Scene View).

- <kbd>WASD</kbd> + Hold and move <kbd>Right Mouse</kbd>: Move around the scene view (like you are controlling player moving around the scene).

- Scroll <kbd>Middle Mouse</kbd> or <kbd>Alt</kbd> + Hold and move <kbd>Right Mouse</kbd>: Zoom in/out.

- <kbd>F</kbd>: Focus on current selected Game Object.

- <kbd>Ctrl</kbd> + Move Game Object: Move Game Oject by fixed distance (0.25)
