# Unity releases version meaning

There are 2 types of Unity releases

- Long-Term Support (LTS) releases
- TECH Stream releases, including alpha and beta releases

Release numbers mainly consist of the year plus a minor release number: 1, 2, or 3.
There are two TECH stream releases per year (2021.1 and 2021.2) and one Long-Term Support release (2021.3 LTS).

**Note:** In 2018 and 2019, there were three TECH stream releases and one LTS release.
